## Repository Structure

```
├── data/                        # Input datasets
│   ├── raw_data_thesis.csv      # Raw market data (SOFR, EFFR, spreads, reserves, TGA)
│   ├── cross_currency_basis.csv # JPY/USD cross-currency basis
│   └── data_master.csv          # Cleaned panel (generated by step1)
│
├── R/                           # Analysis scripts (run sequentially)
│   ├── config.R                 # Path configuration (sourced by all scripts)
│   ├── step1_data_preparation.R # Data cleaning, variable construction, ADF tests
│   ├── step2_granger_causality.R# Granger causality test batteries
│   ├── step3_var_irf.R          # VAR estimation, IRFs, FEVD (VARs A–D)
│   └── step4_regime_analysis.R  # Regime-dependent analysis (5 bp threshold)
│
├── reports/                     # R Markdown sources and compiled PDFs
│   ├── report_data_preparation.Rmd / .pdf   # Section 5.2
│   ├── report_granger.Rmd / .pdf            # Section 5.3
│   ├── report_var.Rmd / .pdf                # Section 5.4
│   └── report_regime.Rmd / .pdf             # Section 5.5
│
└── output/
    ├── figures/                 # PDF figures (generated by R scripts)
    ├── tables/                  # LaTeX tables (generated by R scripts)
    ├── granger_all_results.csv  # Full Granger test results
    └── granger_regime_results.csv # Regime-conditional Granger results
```

## Replication

Run from the repository root:

```r
source("R/step1_data_preparation.R")  # ~2 min
source("R/step2_granger_causality.R") # ~1 min
source("R/step3_var_irf.R")          # ~5 min (bootstrap IRFs)
source("R/step4_regime_analysis.R")  # ~1 min
```

Reports can be rebuilt with:

```r
rmarkdown::render("reports/report_data_preparation.Rmd")
rmarkdown::render("reports/report_granger.Rmd")
rmarkdown::render("reports/report_var.Rmd")
rmarkdown::render("reports/report_regime.Rmd")
```
